# LocalStyle Brand Management Dashboard

A comprehensive brand management dashboard built with Next.js 15, React 19, and TypeScript. This application provides a complete solution for managing clothing brands with features including product management, category organization, staff management, and analytics.

## Features

- **Product Management**: Create, edit, and organize products with images, pricing, and inventory
- **Category System**: Hierarchical category management with SEO optimization
- **Staff Management**: Role-based access control with invite system
- **Brand Profile**: Customizable brand information and settings
- **Dashboard Analytics**: Key metrics and performance indicators
- **Responsive Design**: Works on all device sizes
- **Dark Mode**: Modern dark theme with OKLCH color system
- **Authentication**: Secure login system with role-based permissions

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Demo Login:
- Username: Any username (min 3 characters)
- Password: Any password (min 6 characters)

## Technology Stack

- **Framework**: Next.js 15 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS with OKLCH color system
- **State Management**: Redux Toolkit
- **Data Fetching**: React Query
- **Form Handling**: React Hook Form with Zod validation
- **UI Components**: Radix UI and custom components
- **Icons**: Lucide React

## Deployment

To deploy this application:

1. Build the project:
   ```bash
   npm run build
   ```
2. Start the production server:
   ```bash
   npm start
   ```

The application can be deployed to any platform that supports Node.js, including Vercel, Netlify, or traditional hosting providers.
